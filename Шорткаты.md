**1\. НАВИГАЦИЯ И ПОИСК ПО КОДУ**

**Переход к определениям и реализации**

| Сочетание | Действие | Применение |
| :---- | :---- | :---- |
| **⌘ \+ B** | Перейти к объявлению (Declaration) | Быстрый переход к определению класса, функции или переменной под курсором |
| **⌥ \+ ⌘ \+ B** | Перейти к реализации (Implementation) | Навигация к конкретной реализации, если существует несколько вариантов |
| **⇧ \+ ⌘ \+ B** | Перейти к типу объявления | Переход к типу, на который ссылается переменная |

**Поиск везде**

| Сочетание | Действие | Применение |
| :---- | :---- | :---- |
| **⇧ \+ ⇧** (двойное нажатие) | Search Everywhere | Универсальный поиск по файлам, классам, символам и содержимому проекта |
| **⌘ \+ O** | Go to Class | Поиск класса по имени |
| **⇧ \+ ⌘ \+ O** | Go to File | Поиск файла по имени |
| **⌥ \+ ⌘ \+ O** | Go to Symbol | Поиск функций, методов, атрибутов классов |

**Поиск текста в файлах**

| Сочетание | Действие | Применение |
| :---- | :---- | :---- |
| **⌘ \+ F** | Find in File | Поиск текста в текущем файле |
| **⇧ \+ ⌘ \+ F** | Find in Project | Поиск строки по всему проекту, папкам или модулям |
| **⌘ \+ F / ⌘ \+ R** | Find / Replace in File | Поиск и замена в текущем файле |
| **⇧ \+ ⌘ \+ R** | Replace in Project | Замена во всём проекте |
| **⌘ \+ G / ⇧ \+ ⌘ \+ G** | Next / Previous Occurrence | Навигация к следующему и предыдущему совпадению |

**Поиск использования (Uses)**

| Сочетание | Действие | Применение |
| :---- | :---- | :---- |
| **⌥ \+ F7** | Find Usages | Найти все места использования символа в проекте |
| **⌘ \+ F7** | Find Usages in File | Найти использование в текущем файле |
| **⇧ \+ ⌘ \+ F7** | Highlight Usages | Подсветить все использования символа в файле |
| **⌥ \+ ⌘ \+ F7** | Show Usages | Показать окно с использованиями |

**СОВЕТЫ ПРОФЕССИОНАЛОВ**

* **Используйте Key Promoter X плагин** — показывает сочетание клавиш вместо кликов мышью.  
* **Отключите мышь во время работы** — используйте только клавиатуру, это ускорит работу в 2-3 раза.  
* **Запомните структуру сочетаний:**  
  * **⌘** \= базовые операции  
  * **⌥** \= дополнительные функции / навигация  
  * **⌃** \= системные действия и запуск  
  * **⌥ \+ ⌘** \= рефакторинг и трансформация

### **ИЗМЕРИМЫЕ ПРЕИМУЩЕСТВА**

* **40-50%** — при написании нового кода  
* **60-70%** — при рефакторинге  
* **50-60%** — при отладке и навигации  
* **30-40%** — общее увеличение производительности.

### **ЗАКЛЮЧЕНИЕ**

Овладение этими сочетаниями на macOS позволит вам кодировать, не отвлекаясь на интерфейс. Рекомендуется начать с базовых 15 сочетаний, ежедневно добавляя по 2-3 новых в свою мышечную память.
